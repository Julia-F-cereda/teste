<h3>DESCRIÇÃO DO PROJETO</h3>
Este projeto foi desenvolvido com o objetivo de validar dados inseridos em um sistema de boletins escolares. A aplicação realiza verificações para garantir que as notas informadas estejam dentro dos critérios estabelecidos, assegurando a integridade e a consistência das informações.
O foco do projeto foi praticar validação de dados e controle de entradas, aspectos fundamentais em testes de software e garantia de qualidade.

<h3>COMO EXECUTAR</h3>
<P>1. Baixe os arquivos do projeto.</P>
<P>2. Abra o excel e insira o plano_de_teste`.</P>
<P>3. baixe o pytest = pip install pytest</P>
<p>4. Rode o arquivo no pytest</p>

<h3>FUNCIONALIDADES DO PROJETO</h3>

<p>Validação de notas inseridas</p>
<p>Verificação de valores mínimos e máximos permitidos</p>
<p>Bloqueio de entradas inválidas</p>
<p>Garantia de integridade dos dados</p>
